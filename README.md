## How to run (local)

```bash
# create & activate venv (optional but recommended)
python -m venv venv
# Windows
venv\Scripts\activate
# macOS/Linux
source venv/bin/activate

# install dependencies (if any)
pip install -r requirements.txt

# run the app
python app.py
```
